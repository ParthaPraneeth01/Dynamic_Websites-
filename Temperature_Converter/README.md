# Temperature Converter

A smart temperature converter web app that converts between Celsius, Fahrenheit, and Kelvin with formula display and conversion history.

## Features
- Convert between Celsius, Fahrenheit and Kelvin
- Shows formula used for each conversion
- Tracks recent conversion history
- Clear history option
- Input validation with error messages
- Reset functionality

## Concepts Used
- JavaScript Variables & Data Types
- parseInt() for type conversion
- if / else if / else conditionals
- DOM Manipulation
- getElementById & textContent
- createElement & appendChild
- onclick Events
